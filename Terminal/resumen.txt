git init -> sirve para inicializar el repositorio, git add . -> sirve para agregar todos los archivos creados o modificados y pasarlos al estado added, git commit -m nombreCommit -> sirve para subir los cambios al staging area, git push origin main -> sirve para subir los archivos commiteados al repositorio en nube, git clone -> es para clonar un repositorio de la nube a mi maquina local
